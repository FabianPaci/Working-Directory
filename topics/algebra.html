Algebra

Index laws

  1. \(a^m \times a^n = a^{m + n}\)
  2. \(a^m \div a^n = a^{m-n}\)
  3. \((a^m)^n = a^{m \times n}\)
  4. \(a^{-m} = \frac {1}{a^m}\)
  5. \(a^0 = 1\)
  6. \(a^{\frac{n}{m}} = \root m \of{a^n}\)

Examples

  1. \(2^3 \times 2^7 = 2^{10}\)
  2. \(\frac{3^6}{3^2} = 3^4\)
  3. \((5^2)^5 = 5^{10}\)
  4. \(7 \times 2^{-2} = \frac{7}{2^2}\)
  5. \(45^0 = 1\)
  6. \(5^{\frac{-3}{7}} = \frac{1}{\root 7 \of {5^3}}\)

Simultaneous equations

Simultaneous equations is used to find two variables with two equations

Substitution

  1. \(2x + 4y = 6\)
  2. \(3y = 3 - 2x\)

Start by solving one variable (\(x\)) from one equation (\(2x + 4y = 6\))

  • \(2x = 6 - 4y\)
  • \(x = \frac{6}{2} - \frac{4}{2}\)
  • \(x = 3 - 2y\)

Then substitute the solution (\(x = 3 - 2y\)) into the other equation (\(3y = 3 - 2x\))

  • \(3y = 3 - 2(3 - 2y)\)
  • \(3y = 3 - 6 + 4y\)
  • \(3y - 4y = 3 - 6\)
  • \(-y = -3\)
  • \(y = \frac{-3}{-1}\)
  • \(y = 3\)

Finally substitute the solved variable \(y = 3\) into any equation \(3y = 3 - 2x\)

  • \(3 \times 3 = 3 - 2x\)
  • \(9 = 3 - 2x\)
  • \(2x = 3 - 9\)
  • \(x = \frac{3}{2} - \frac{9}{2}\)
  • \(x = \frac{11}{2}\)

Therefore express the solutions

  • \(x = \frac{11}{2}\)
  • \(y = 3\)

or

  • \((5.5 , 3)\)

Elimination

  1. \(4y - 3x = 6\)
  2. \(3y - 2x = 3\)

Cross multply the equations to create common factors

  • \(3(4y - 3x) = 3 \times 6\)
  • \(12y -9x = 18\)
  • \(4(3y - 2x) = 4 \times 3\)
  • \(12y - 8x = 12\)

Subtract one variable (\(y\)) so you're left with the other one (\(x\))

  • \(\begin{matrix} 12y - 8x = 12 \\ - 12y - 9x = 18 \\ \hline x = 16 \end{matrix}\)

Substitute the solution (\(x = 16\)) into any equation

  • \(3y = 3 - 2 \times 16\)
  • \(3y = -29\)
  • \(y = \frac{-29}{3}\)

Finally, express the solutions

  1. \(x = 16\)
  2. \(y = \frac{-29}{3}\)

or

  • \((16 , \frac{-29}{3})\)

Logarithms (logs)

\(\log_{(a)}(y) = x <=> a^x = y\)

Log laws

  1. \(log_{(a)}(xy) = \log_{(a)}(x) + \log_{(a)}(y)\)
  2. \(log_{(a)}(\frac{x}{y}) = \log_{(a)}(x) - \log_{a}(y)\)
  3. \(log_{(a)}(x^n) = n\log_{(a)}(x) => \log_{(a)}(\frac{1}{x}) = -\log_{(a)}(x)\)
  4. \(log_{(a)}(a) = 1\)
  5. \(log_{(a)}(1) = 0\)
